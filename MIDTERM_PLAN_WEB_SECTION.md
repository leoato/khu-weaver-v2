# 웹 프로토타입 체험 안내

본 프로젝트의 프로토타입은 웹 환경에서 바로 체험할 수 있도록 배포할 예정이다.

- 체험 URL: `배포 후 Netlify 주소 입력`
- 대상 사용자: 경희대학교 전자정보대 학생 및 심사위원
- 핵심 기능: 학번별 전공 이수체계 로드맵 확인, 이수 과목 반영, 전공 평점/졸업 요건 분석, 진로 트렌드 확인
- OCR 기능: 에브리타임 또는 인포21 성적 스크린샷을 업로드하면 Gemini 기반 OCR 분석을 통해 전공 과목과 성적을 자동 인식하고 로드맵에 반영한다.

API 키 보호 방식은 다음과 같다. 브라우저 코드에는 Gemini API 키를 포함하지 않고, Netlify Function을 서버리스 프록시로 두어 `/api/ocr` 요청만 받는다. 실제 `GEMINI_API_KEY`는 Netlify 환경변수에 저장되며, 서버 함수가 Gemini API를 대신 호출한다. 따라서 심사위원이 웹 프로토타입에서 스크린샷 분석 기능을 사용하더라도 API 키는 클라이언트에 노출되지 않는다.

개인정보 보호를 위해 업로드한 이미지는 서버에 저장하지 않고 분석 요청에만 사용한다. 성적 및 이수 과목 정보는 사용자의 브라우저 로컬 저장소에만 반영되도록 구성한다.
