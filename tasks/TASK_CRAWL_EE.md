# [브리프] 전자공학과 <연도>학번 교육과정 데이터 구축

> 대상 모델: 소넷 · 소요 파일: 이 브리프 + 아래 "읽을 파일" 2개만. **다른 파일은 읽지 마시오(토큰 절약).**
> 사용자가 지정한 연도(2020/2021/2022/2024 중 하나)를 <연도>로 삼아 수행.

## 목표
`khu-weaver_v1/js/data/EE_<연도>.js` 신규 생성 + 검증 PASS + STATUS 보고.

## 절대 규칙
1. **기존 파일은 어떤 것도 수정·삭제 금지.** 산출물은 새 파일 `EE_<연도>.js` 하나뿐.
2. registry.js, index.html 건드리지 않기 (통합 세션이 일괄 처리).
3. 데이터를 추측으로 채우지 않기 — 반드시 PDF 원문 근거. 불확실하면 STATUS에 "특이사항"으로 기록.

## 읽을 파일 (이 2개만)
1. `khu-weaver_v1/docs/DATA_SCHEMA.md` — 스키마 기준 (전부 읽기)
2. `khu-weaver_v1/js/data/EE_2026.js` — 형식 참고용 예시 (전부 읽기)

## 수행 절차

### 1단계: PDF 확보 (web_fetch 사용, bash curl은 차단되어 있음)
- 전자공학부 홈페이지 교육과정 게시판: `https://ee.khu.ac.kr` → 게시판 코드 BMSR00048, menuNo=21700025
- 게시판 목록에서 "<연도>학년도 전자공학과 교육과정" 게시글 탐색 (2020~2026 PDF 7개 존재 확인됨)
- 게시글은 POST(view())로 열리며, 첨부 다운로드 URL 패턴: `cmmn/file/fileDown.do?atchFileId=...&fileSn=1`
- **web_fetch로 PDF URL을 직접 fetch하면 텍스트 추출됨** (v1.5에서 검증된 방법)
- 필요한 부분: [별표1] 편성표(과목/학수번호/학점/학년/학기), [별표3] 학년별 편성표(동시개설 과목의 주개설학기), [별표4] 선수과목 지정표, [별표2] 전공심화 분류, 졸업이수학점 조항(제11조 부근)

### 2단계: 데이터 파일 작성
- DATA_SCHEMA.md 규격대로 `js/data/EE_<연도>.js` 작성. EE_2026.js와 동일한 스타일(상단 출처 주석, 섹션 주석, 선수과목 한글 주석) 유지.
- 주의: 학번마다 과목 구성이 다름. 2026 파일을 복사해서 고치는 방식 금지 — PDF 기준으로 처음부터 작성.

### 3단계: 검증 (필수)
```bash
cd "khu-weaver_v1" && node tools/validate_curriculum.js js/data/EE_<연도>.js
```
- FAIL이면 수정 후 재검증. PASS까지 반복.
- 출력된 이수구분별 학점 합계를 PDF [별표1] 합계와 눈으로 대조.

### 4단계: 보고
`tasks/STATUS.md` 맨 아래에 **자기 줄 1줄만 추가** (다른 줄 수정 금지):
```
| C-<연도> | EE_<연도>.js | 과목 N개 / 선수 N건 | PASS | (특이사항 또는 -) | YYYY-MM-DD |
```
