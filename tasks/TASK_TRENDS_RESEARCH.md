# [브리프] R-1: 진로 트렌드 실데이터 리서치 (모델: 소넷 + 웹서치)

> 발행: 페이블 2026-07-07 · 산출물: `khu-weaver_v2/js/data/TRENDS.js` 신규 1개 (다른 파일 수정 금지)
> **절대 규칙: 수치 창작 금지.** 현재 사이트의 92%/86% 등은 데모용 창작 수치라 교체가 목적. 출처를 확인할 수 없는 수치는 싣지 않는다. 항목이 3개뿐이어도 출처 있는 3개가 창작 8개보다 낫다.

## 수집 대상 (우선순위순, 웹서치 사용)
1. 전자·전기공학 계열 졸업생 취업 통계: 대학알리미(경희대 전자공학과 공시 취업률), 교육부/한국교육개발원 취업통계연보
2. 산업별 인력 수요: 반도체(산업통상자원부·한국반도체산업협회 인력 전망), AI/SW(고용노동부·한국고용정보원 중장기 인력수급 전망), 임베디드/모빌리티
3. 가능하면 "단기간 수요 증가" 근거가 되는 연도별 비교 수치

## 산출 포맷 (TRENDS.js)
```js
window.KW_TRENDS = {
    updated: "2026-07",
    items: [ // 세로 막대그래프용. value 단위·기준을 명확히
        { label: "반도체 인력 수요", value: 00, unit: "%", basis: "2031년까지 연평균 증가 전망", source: "한국반도체산업협회 (2024)", url: "https://..." },
        ...
    ],
    analysis: [ // 5줄 내외. 각 줄은 items의 수치·출처로 뒷받침 가능한 문장만
        "최근 전자공학과 졸업생 취업 분야는 ○○가 가장 높았다 (출처: ...).",
        ...
    ]
};
```

## 검증·보고
- 각 수치에 url 필수. url 접속 확인(web_fetch)까지 하고 STATUS.md에 1줄(| R-1 | TRENDS.js | 항목 N개 | 전 출처 링크 확인 | 특이사항 | 날짜 |).
- node --check TRENDS.js. index.html 등록은 하지 말 것(F-3 세션이 처리).
