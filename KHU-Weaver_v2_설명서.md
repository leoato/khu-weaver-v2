# KHU-Weaver v2 — 팀원용 요약 설명서

> 팀 공모자들 (차동균·박재홍·이원진) · 제5회 전자정보대학 창의혁신 공모전 출품작 · 2026-07-08 기준

## 한 줄 소개
OP.GG가 게임 데이터를 분석해 최적 빌드를 추천하듯, 학교 공식 교육과정 데이터로 전자공학과 학생의 '전공 이수 빌드'를 시각화·추천하는 웹 플랫폼.

## 실행 방법
- `khu-weaver_v2/index.html`을 브라우저(크롬 권장)로 열면 끝. 설치·서버 불필요.
- 입력한 정보(이수 과목·성적)는 그 브라우저에만 저장되고 외부로 전송되지 않음.

## 사용 흐름 (첫 접속 기준)
1.  온보딩 3단계 : 학과 선택 → 학번·이름 → 이수 이력 입력 방법 선택
2. 이수 이력은 3가지 중 하나로:
   - 📷  에타 성적 스크린샷 업로드 (최대 8장) → Gemini AI가 과목·성적을 자동 인식해 반영
   - ✅  직접 선택  — 학년별 4열 표에서 체크 (학년별 "전체 선택" 버튼 있음)
   - 🌱  신입생 스킵  — 바로 로드맵으로
3.  메인 로드맵 : 1~4학년 과목이 노드로 표시. 상태 3종 — 이수완료(버건디+✓) / 수강가능(파란 테두리) / 선수과목 미충족(회색 점선+🔒). 노드 클릭 시 선후수 연결이 하이라이트로 고정됨.
4.  트랙 탭 : 반도체·임베디드·AI신호·통신·회로 트랙별로 추천 과목이 강조됨. 선배 빌드 프리셋 버튼도 있음.
5.  진로 트렌드 탭 : 취업률·인력수요 통계(전 항목 공식 출처+링크 표기) + 분석 5줄.
6.  내 분석 탭 : 졸업 이수율(130학점 기준), 전공 평점(4.3 만점, 성적 입력한 과목만 반영), 역량 레이더.

## 데이터가 믿을 만한 이유 (심사 어필 포인트)
- 전자공학부 공시 교육과정 PDF(별표1·2·3·4)에서 구축한 실데이터 —  2020~2026 전 학번 7개  커버 (학번당 51~52과목, 선수과목 약 40건)
- 표준 스키마 + 자동 무결성 검증 스크립트로 품질 관리 (`docs/DATA_SCHEMA.md`, `tools/validate_curriculum.js`)
- 진로 트렌드 수치도 전부 출처 링크 보유 (창작 수치 없음)

## 팀원이 알아둘 것
-  AI 스크린샷 인식을 쓰려면  `khu-weaver_v2/js/config.js`에 Gemini API 키가 있어야 함. 키가 비어 있으면 해당 기능만 꺼지고 나머지는 정상 동작.  키가 든 파일은 절대 외부 공유 금지. 
- 폴더 구조: `khu-weaver_v1/` = 리디자인 전 백업, `khu-weaver_v2/` =  현재 정본 . 수정은 v2에서만.
- 버전 이력은 `khu-weaver_v2/PATCHNOTES.md`, 남은 할 일은 `khu-weaver_v2/ROADMAP.md`와 `tasks/` 참조.
